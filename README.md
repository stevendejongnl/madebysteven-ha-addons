# Made By Steven - Home Assistant Addons

A collection of custom Home Assistant addons.

## Installation

1. Go to **Settings → Add-ons → Add-on Store**
2. Click ⋮ → **Repositories**
3. Add: `https://github.com/stevendejongnl/madebysteven-ha-addons`

## Addons

### Picnic Swipe
Integrates the Picnic grocery delivery app with a swipe UI to add products to your basket.
